## Configurable Variables

The default variable of env.

```
ENV TW_PORT=8080 \
  TW_ROOTTIDDLER=$:/core/save/all \
  TW_RENDERTYPE=text/plain \
  TW_SERVETYPE=text/html \
  TW_USERNAME="" \
  TW_PASSWORD="" \
  TW_HOST=0.0.0.0 \
  TW_PATHPREFIX=""
```

The alternative scheme is replace the default command.
